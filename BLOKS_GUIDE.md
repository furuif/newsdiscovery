## 1. 布鲁可品牌

### 1.1 公司简介

**布鲁可 (Bloks)** 是中国领先的积木玩具品牌，由**上海布鲁可积木科技有限公司**运营。

- **成立时间**: 2014 年
- **总部**: 中国上海
- **创始人**: 朱伟松
- **品牌定位**: 专注于儿童益智积木，强调"为中国儿童设计"

**公司背景**:
- 布鲁可的前身是"葡萄科技"，最初从事儿童智能硬件
- 2015 年正式推出布鲁可积木品牌
- 2020 年后快速发展，成为中国积木市场的重要参与者
- 与多家知名 IP 合作（奥特曼、变形金刚、火影忍者等）

### 1.2 与 LEGO 的区别

| 对比项 | 布鲁可 (Bloks) | 乐高 (LEGO) |
|--------|---------------|-------------|
| **起源** | 中国 (2014) | 丹麦 (1932) |
| **目标市场** | 主要面向中国市场，兼顾全球 | 全球市场 |
| **价格定位** | 中端性价比 | 高端溢价 |
| **IP 合作** | 奥特曼、变形金刚、火影等日系/美系 IP | 迪士尼、漫威、星战等 |
| **零件系统** | 独立规格，部分兼容 LEGO | 自有标准，行业标杆 |
| **设计理念** | 大颗粒起步，注重低龄儿童安全 | 全年龄段覆盖 |
| **编程教育** | 有编程积木系列 | LEGO Education/MINDSTORMS |

### 1.3 主要产品线

布鲁可的产品线主要分为以下几类：

#### 按年龄/颗粒大小分类

1. **大颗粒系列** (适合 3-6 岁)
   - 启蒙积木
   - 大颗粒拼插
   - 强调安全性和易抓握

2. **小颗粒系列** (适合 6 岁+)
   - 标准积木
   - 复杂模型构建
   - 更高自由度

3. **科技机械系列**
   - 齿轮、电机、传动件
   - 可编程积木
   - STEM 教育导向

#### 按 IP/主题分类

1. **原创 IP**
   - 布鲁可战队
   - 百变布鲁可
   - 星际探险系列

2. **授权 IP**
   - 奥特曼系列 (人气最高)
   - 变形金刚
   - 火影忍者
   - 航海王
   - 宝可梦

3. **教育系列**
   - 编程机器人
   - STEM 科学套装
   - 学校/机构定制

## 2. 星辰版系列

### 2.1 星辰版是什么产品线

**星辰版**是布鲁可推出的**高自由度创造系列**，定位为：

- **高端创造线**: 面向进阶玩家和 MOC (My Own Creation) 爱好者
- **高零件密度**: 单盒零件数量多，可搭建多个模型
- **通用性强**: 零件设计注重通用性和复用性

### 2.2 特色零件/设计

星辰版的特色包括：

1. **特殊连接件**
   - 多向连接轴
   - 万向关节
   - 弹性连接件

2. **科技件**
   - 齿轮组 (多种齿数比)
   - 差速器
   - 转向机构
   - 悬挂系统

3. **装饰件**
   - 印刷件 (非贴纸)
   - 透明件 (多种颜色)
   - 发光件 (部分套装)

4. **结构件**
   - 梁 (多种长度和孔位)
   - 销 (摩擦销、光销)
   - 轴套

### 2.3 目标用户群体

- **年龄**: 8 岁及以上
- **技能水平**: 有积木搭建经验的进阶玩家
- **兴趣方向**:

## 3. 积木规格

### 3.1 零件尺寸规格

布鲁可积木采用**独立规格系统**，但设计上参考了行业标准：

#### 基础砖块

- **1×1 砖**: 约 8mm × 8mm × 9.6mm (含凸点)
- **2×4 砖**: 约 16mm × 32mm × 9.6mm (标准砖)
- **板 (Plate)**: 厚度约 3.2mm (1/3 砖高)

#### 技术件

- **梁 (Beam)**: 孔间距约 8mm
- **轴 (Axle)**: 直径约 4.8mm，十字截面
- **销 (Pin)**: 直径约 4.8mm

> **注意**: 以上尺寸为近似值，实际测量可能有细微差异。布鲁可的公差控制与 LEGO 不同。

### 3.2 连接方式

布鲁可积木采用多种连接方式：

1. **凸点 - 孔洞系统 (Stud-and-Tube)**
   - 顶部凸点 (Stud)
   - 底部管状孔 (Tube)
   - 摩擦力固定

2. **销连接 (Pin Connection)**
   - 摩擦销 (Friction Pin)
   - 光销 (Smooth Pin)
   - 用于技术件连接

3. **轴连接 (Axle Connection)**
   - 十字轴
   - 配合齿轮、轮子等

4. **卡扣连接 (Clip Connection)**
   - 用于面板、装饰件
   - 可拆卸设计

### 3.3 兼容性

#### 与 LEGO 的兼容性

| 兼容性类型 | 程度 | 说明 |
|-----------|------|------|
| **物理兼容** | ⚠️ 部分兼容 | 基础砖块可拼接，但松紧度可能不同 |
| **技术件兼容** | ⚠️ 有限兼容 | 轴、销尺寸接近，但公差不同 |
| **人仔兼容** | ❌ 不兼容 | 布鲁可人仔比例和接口不同 |
| **底板兼容** | ⚠️ 部分兼容 | 凸点间距相近，但边缘可能不齐 |

**兼容性注意事项**:
- 布鲁可积木的凸点可能略松或略紧
- 长期混用可能导致零件磨损
- 建议主要使用同品牌零件搭建核心结构
- 装饰件、非承重件可以混用

### 3.4 特殊零件类型

布鲁可的特色零件包括：

1. **球形关节**
   - 多自由度连接
   - 用于人偶、机械臂

2. **软胶件**
   - 轮胎
   - 履带
   - 柔性连接

3. **电子件**
   - 电机 (多种规格)
   - 传感器 (红外、触碰、颜色)
   - 控制 hub

4. **印刷件**
   - 直接印刷图案 (非贴纸)
   - 耐久性好

5. **透明件**
   - 透明砖
   - 发光导光件
   - 彩色透明片

## 4. 设计软件

### 4.1 官方 3D 设计软件

截至 2024 年，布鲁可**尚未推出官方的通用 3D 设计软件**（类似 LEGO Digital Designer 或 Stud.io）。

**现状**:
- 布鲁可主要提供**编程软件**用于可编程积木
- 搭建指导以**纸质说明书**和**APP 电子说明书**为主
- 官方 APP 提供部分模型的 3D 预览

### 4.2 编程软件

布鲁可为可编程积木系列提供编程环境：

1. **布鲁可编程 APP**
   - 平台：iOS / Android
   - 语言：图形化编程 (Scratch 风格)
   - 功能：
     - 图形化积木编程
     - 实时调试
     - 作品分享

2. **PC 编程软件**
   - 部分套装支持 PC 端
   - 基于 Scratch 3.0 修改
   - 支持 Python 扩展 (部分型号)

### 4.3 文件格式

由于缺乏官方设计软件，布鲁可**没有公开的标准文件格式**。

**已知格式**:
- 编程项目文件：`.bcp` (布鲁可编程格式，私有)
- 固件文件：`.bin` (用于更新控制 hub)

### 4.4 API 或数据导出

**官方 API**: 
- ❌ 目前没有公开的官方 API
- ❌ 没有零件数据库导出功能

**替代方案**:
- 社区正在尝试逆向工程
- 部分爱好者手动整理零件清单
- 可使用通用 3D 建模软件 (Blender, Fusion 360) 自行建模

### 4.5 第三方设计工具

由于缺乏官方工具，MOC 创作者通常使用：

1. **BrickLink Studio / Stud.io**
   - 可导入自定义零件
   - 需要手动创建布鲁可零件库
   - 渲染效果好

2. **Blender**
   - 开源 3D 建模
   - 需要自行建模或使用社区模型
   - 灵活但学习曲线陡

3. **OpenSCAD**
   - 参数化建模
   - 适合技术件设计
   - 代码驱动

## 6. 3D 打印相关

### 6.1 布鲁可零件的 3D 打印可行性

**可行性评估**: ✅ **可行，但有挑战**

**优势**:
- 零件几何形状相对规则
- 社区有分享打印文件
- 可用于补充稀有零件或定制零件

**挑战**:
- 官方未提供 STL 文件
- 需要自行测量和建模
- 公差控制要求高

### 6.2 公差要求

布鲁可积木的公差控制较为严格：

| 配合类型 | 推荐公差 | 说明 |
|---------|---------|------|
| **凸点 - 孔洞** | ±0.05mm | 过松易脱落，过紧难拆卸 |
| **销 - 孔** | ±0.03mm | 摩擦销需要精确配合 |
| **轴 - 孔** | ±0.05mm | 十字轴需要转动顺畅 |
| **卡扣** | ±0.1mm | 需要弹性变形空间 |

**3D 打印建议**:
- **打印机精度**: 建议使用 0.1mm 层高的 FDM 或光固化
- **材料选择**: 
  - PLA: 易打印，但长期可能变形
  - PETG: 韧性好，适合连接件
  - ABS: 强度高，但需要加热床
  - 树脂 (光固化): 精度高，适合小零件
- **缩放补偿**: 可能需要 0.5-1% 的缩放补偿

### 6.3 连接强度注意事项

**承重考虑**:
- 3D 打印零件的层间结合力较弱
- 避免在受力方向垂直于打印层
- 关键承重件建议增加壁厚 (≥2mm)

**耐磨性**:
- 频繁拆卸的零件容易磨损
- 建议打印多份备用
- 可考虑尼龙材料提高耐磨性

**长期稳定性**:
- PLA 在高温环境下可能变形
- 避免阳光直射和高温环境
- 定期检查连接松动情况
