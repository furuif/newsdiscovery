## 🔑 配置提醒

需要配置 `.env` 文件以启用真实功能：

```bash
# Qwen-VL API 配置（百炼）
QWEN_API_KEY=your_api_key_here
QWEN_MODEL=qwen-vl-max

# 拓竹打印机配置（可选）
BAMBU_PRINTER_ID=
BAMBU_ACCESS_CODE=
BAMBU_CONNECTION=lan
```

获取 API Key: https://dashscope.console.aliyun.com/
