## 🐛 已知问题

### 1. 3D 预览显示占位模型
**现象**: 显示立方体而非真实模型  
**原因**: STL 加载器未实现  
**影响**: 低 - 不影响核心功能  
**解决**: 添加 `three-stdlib` 依赖

### 2. 文件下载按钮无响应
**现象**: 点击下载无反应  
**原因**: 下载逻辑未实现  
**影响**: 中 - 无法获取 STL 文件  
**解决**: 实现文件下载 API

### 3. WebSocket 断线后需刷新
**现象**: 断线后进度停止  
**原因**: 自动重连未实现  
**影响**: 低 - 刷新页面可恢复  
**解决**: 添加重连逻辑
